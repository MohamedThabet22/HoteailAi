# 🏨 Hotelio AI - Luxury Hotel Booking Experience

[![Platform](https://img.shields.io/badge/Platform-Android-green.svg)](https://developer.android.com)
[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-purple.svg)](https://kotlinlang.org)
[![Compose](https://img.shields.io/badge/Jetpack_Compose-Material_3-blue.svg)](https://developer.android.com/jetpack/compose)
[![Firebase](https://img.shields.io/badge/Firebase-Supported-orange.svg)](https://firebase.google.com)

**Hotelio AI** is a premium, high-performance Android application designed for travelers who seek a regal and seamless hotel booking experience. Built with modern Android development practices, it combines luxury aesthetics with powerful features.

---

## ✨ Key Features

### 🌟 Elite UI/UX Experience
*   **Luxury Design Language:** A sophisticated interface inspired by high-end hospitality brands.
*   **Fluid Animations:** Seamless transitions, scaling effects, and animated visibility for a "living" UI.
*   **Dark/Light Support:** Fully optimized for Material 3 design systems.

### 📅 Advanced Booking System
*   **Dynamic Pricing:** Real-time calculation based on duration, guest count, and service fees.
*   **Smart Date Selection:** Integrated modern DatePicker with duration management.
*   **Trips History:** Keep track of all your past and upcoming regal stays in a dedicated dashboard.

### 🗺️ Intelligent Integration
*   **Live Google Maps:** Interactive maps for every hotel to explore locations instantly.
*   **Real-time Sync:** Powered by Firebase to ensure your favorites and bookings are always up-to-date across devices.
*   **Collections:** Curated categories (Resorts, Beach, Villas) with filtered discovery.

### 🔒 Security & Reliability
*   **Secure Checkout:** SSL-encrypted payment flows and PCI-compliant UI.
*   **Offline Support:** Powered by Room Database for smooth performance even with poor connectivity.

---

## 🛠 Tech Stack

*   **Language:** [Kotlin](https://kotlinlang.org/)
*   **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (100%)
*   **Architecture:** MVVM (Model-View-ViewModel) + Clean Architecture principles.
*   **Dependency Injection:** [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
*   **Local Database:** [Room](https://developer.android.com/training/data-storage/room)
*   **Backend & Auth:** [Firebase](https://firebase.google.com/) (Auth, Firestore, Realtime Database)
*   **Map Integration:** [Google Maps Compose](https://github.com/googlemaps/android-maps-compose)
*   **Image Loading:** [Coil](https://coil-kt.github.io/coil/)
*   **Animations:** [Lottie](https://airbnb.io/lottie/) & Compose Animations API.

## 🚀 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MohamedThabet22/HoteailAi.git
    ```
2.  **Add Firebase:**
    *   Place your `google-services.json` in the `app/` directory.
3.  **Google Maps API:**
    *   Add your API key to `local.properties` or `AndroidManifest.xml`.
4.  **Build & Run:**
    *   Open in Android Studio (Ladybug or newer) and run on an emulator or physical device.
